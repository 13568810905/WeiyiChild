  基于Android的幼儿园管理系统,脱机Demo  
## 设计细节
* `TabLayout + ViewPager` 主页面切换
* `GridView` 实现功能列表
* `RecyclerView` 实现信息加载、刷新，含空数据提示
* `GreenDao` 数据库管理
* 多语言设计（Demo中只包含中英文，可自由增加其它语言）
* 动态权限管理
* 多种类型的 `Toast` 提示
* 可完全自定义的日历，可以实现各种变态需求
* 全 `.xml` 矢量图标
* 预置可动态添加的测试数据
